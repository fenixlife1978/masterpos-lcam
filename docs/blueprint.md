# **App Name**: LicoPOS Gold

## Core Features:

- Barcoder Hardware Integration: System-wide USB barcode reader capture logic using keypress timeout events for instant item addition and automated field population.
- Intelligent Inventory Forecast Tool: An AI-powered reasoning tool that analyzes sales trends to predict stockouts and suggest optimal restock levels for liquor items.
- Multi-Currency Ledger: Real-time BS/USD dual pricing throughout the checkout and inventory process with a user-adjustable exchange rate.
- Advanced Customer Debt CRM: Comprehensive tracking of customer balances (Cuentas por Cobrar) with automated debt chronological aging and partial payment logic.
- Virtualized Cash Vault: End-of-day register management including opening balances, transaction histories, and detailed breakdown by payment method.
- Local Persistence Engine: High-reliability client-side data management ensuring zero loss between sessions via robust state persistence.
- Financial Document Generator: Automated PDF generation for accounts receivable and register closure reports using embedded export logic.

## Style Guidelines:

- Primary color: Golden Amber (#D4A017) to represent high-end spirit luxury and clarity.
- Background color: Obsidian Charcoal (#0C0B0A), a heavily desaturated dark hue providing a deep cinematic depth.
- Accent color: Terracotta Flame (#CC4317), analogous to the gold, used for high-impact notifications and transaction totals.
- Recommendation: 'Playfair Display' for headlines to evoke classic craftsmanship, paired with 'DM Sans' for a precise and contemporary UI reading experience. Note: currently only Google Fonts are supported.
- Use the Font Awesome 6 Pro 'Light' weights for icons, emphasizing sharp technicality and high-end retail elegance.
- A sophisticated three-column terminal layout for the POS that collapses gracefully using CSS Grid and flex-containers for professional utility.
- Micro-interactions focused on 'Atmospheric Drift', including pulsating barcode indicators and subtle orbital floating spheres for the idle state.
